This plugin allows the user to load the oak tree academy links into their website via a gutenberg block that takes into account year groups from a dropdown.

== Description ==
This plugin relys on Oak National Academy as a 3rd party all of the links are pointed directly towards their site. The URL for this service is https://www.thenational.academy/ there are no terms and conditions available at the time of publishing but the closest information for the intended audience would be https://www.thenational.academy/information-for-teachers

== Installation ==
Simply activate the plugin then head to your page of choice and select the iTCHYROBOT: Oak Academy Resouce block to get started.

== Upgrade Notice ==
Upgrade as soon as possible as we will keep an eye for any educational changes on the Oak Academy site.
